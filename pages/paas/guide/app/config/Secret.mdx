# Secret

secret 是用于存储配置数据的密钥。

![HhYGbeNqyoYKa7xOJgWcwD8YnLc](/static/HhYGbeNqyoYKa7xOJgWcwD8YnLc.png)

## **创建步骤**

1. **选择集群**：在创建 Secret 之前，首先需要选择目标 Kubernetes 集群。
2. **选择命名空间**：选择 Secret 将要存放的命名空间。
3. **填写密钥名称**：输入一个名称来标识新的 Secret。
4. **添加配置项**：输入配置项的键和值，键是配置项的名称，值是配置项的具体内容。
5. **增加配置项**：如果需要更多的配置项，点击【增加配置项】来添加额外的键值对。
6. **创建**：完成所有配置项的添加后，点击【创建】按钮来生成 Secret。
