# ConfigMap

ConfigMap 是用于存储配置数据的 Kubernetes 资源。

![LaCgbSSkTobkizxAAIhc8NeHnte](/static/LaCgbSSkTobkizxAAIhc8NeHnte.png)

## **创建步骤**

1. **选择集群**：在页面上选择 ConfigMap 将要部署的 Kubernetes 集群。
2. **选择命名空间**：从下拉列表中选择 ConfigMap 将要存放的命名空间。
3. **填写配置名称**：输入新 ConfigMap 的名称，这将作为 ConfigMap 的唯一标识。
4. **添加配置项**：输入配置项的键和值，键是配置项的名称，值是配置项的具体内容。
5. **增加配置项**：如果需要更多的配置项，点击【增加配置项】来添加额外的键值对。
6. **创建**：完成所有配置项的添加后，点击【创建】按钮来生成 ConfigMap。
